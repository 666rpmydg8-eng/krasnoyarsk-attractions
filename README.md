Веб-сайт о достопримечательностях Красноярского края
# Достопримечательности Красноярского края
Проект представляет собой информационный веб-сайт, посвященный достопримечательностям, природным красотам и культурному наследию Красноярского края. Сайт разработан с целью познакомить широкую аудиторию с уникальными местами региона и предоставить удобный способ поиска информации об интересующих объектах.
## О проекте
Сайт содержит описания ключевых достопримечательностей Красноярского края, сгруппированных по категориям: природа, музеи, храмы, города и исторические объекты. Проект создан для популяризации туристического потенциала региона и предлагает пользователям интуитивно понятный интерфейс для навигации.
## Возможности
- Адаптивный дизайн
- Современный UI/UX
- Интерактивные элементы
- Плавный пользовательский интерфейс
- Быстрый деплой через Netlify
## Технологии
*   **HTML5**: Структура веб-страницы.
*   **CSS3**: Стилизация и адаптивный дизайн (файл `styles.css`).
*   **JavaScript**: Реализация фильтрации, поиска и динамического обновления контента.
*   **Netlify**: Хостинг и деплой (исходный URL: `https://imaginative-douhua-f13542.netlify.app/`).
